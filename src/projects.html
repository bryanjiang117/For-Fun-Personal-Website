Videogame Remake

Teamfight Tactics

I recreated Teamfight Tactics, a popular game by Riot Games, using Java GUI. Over 2 months, my friend Eric and I wrote thousands of lines of code through Github, learning as we went. These skills would later make learning JavaScript a breeze.

I replicated the main gameplay: 20+ characters complete with stats; rarities; and uniquely animated abilities, a fixed character pool, both character and player level-up system, and of course the signature auto-battle board.

The focus was OOP, but I learned a ton of skills along the way. Futhermore, despite the messy code and mysterious bugs, this was the first time I experienced the spellbinding excitement of bringing a project to life from scratch.
